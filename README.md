# Strategic-Finance-Patterns-17 (SFAP-17)

簿記2級の知識をベースに、企業の決算書から「本業の実態」と「未来の勝機」を暴く17の財務分析パターン・マスタ。

## 概要
本プロジェクトは、単なる会計処理の効率化（自動仕訳等）ではなく、財務データから経営戦略上の「歪み」を検知し、投資判断やM&Aデューデリジェンスの精度を劇的に向上させるインテリジェンス・ツールです。

## 分析実績
### 任天堂（2026年3月期）
- **判定**: `I_ハードウェア販売型` ＋ `N_収穫期・Ｖ字回復型`
- **分析**: 市場が「減収」により失望売りをする中、マスタは「ハード普及一巡による利益の質（デジタル比率）の向上」を検知。次年度のV字回復を予測。

## 17の判定パターン（抜粋）
- **Q_SNS注目・実態不透明型**: 特別利益による化粧決算の検知。
- **P_大底・キャッシュフロー逆行型**: 表面上の赤字に隠れた真のキャッシュ創出力を特定。
- **G_自転車操業型**: 資産売却と借入の連動による倒産リスクの早期警戒。
